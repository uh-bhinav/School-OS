# School-OS Landing Page Themes Documentation

This document provides a comprehensive overview of all the landing page themes created for the School-OS ERP system. Each theme is designed with specific audiences, aesthetics, and use cases in mind.

We have created **8 distinct themes** that cater to different preferences, cultural contexts, and professional requirements:

1. **Nature-Inspired Light Theme**

4. **Sophisticated Gray Theme**
